## Project Description
This project demonstrates DOM manipulation, event handling, and form handling in JavaScript. It includes practice exercises and a fully interactive to-do list application.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Add tasks
- Delete tasks
- Mark tasks as complete
- Filter tasks
- Clear completed tasks
- DOM manipulation
- Event delegation
  

## How to Run
1. Clone the repository
2. Open index.html in your browser

## Lessons Learned
- DOM selection and traversal
- Event listeners
- Event delegation
- Dynamic content creation

## Challenges Faced
- Filtering tasks
- Updating the task count correctly
- Deleting dynamically created tasks
